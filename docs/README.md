# 職務経歴書

## 概要

フロントエンドエンジニアとして新卒から現在までの4年弱実務経験があります\
実務では人材紹介企業とエンタメ企業に常駐し、大小様々なUI改修、LP制作、ページリニューアル、CMS開発、パフォーマンス改善などを経験しました\
個人開発でも主にReactやNext.js, TypeScriptを用いてアプリを開発しています

## 詳細

### 株式会社メンバーズ（2022/04 - 現在）

2022/7 - 2025/3まで人材紹介企業のデザイン部署にフロントエンドエンジニアとして常駐\
2025/9 - エンタメ企業でフロントエンドエンジニアとして常駐

#### サイト制作

- サイト制作では運用や保守を考慮し、要素が増減したときやテキストが複数行になったとき、アニメーションはどうするかなどのデザインデータに記載のない仕様の部分を着手前に拾いながら進めた
- PSIスコア改善のため主にFCPとLCPの改善に取り組み、ファーストビューより下の画像の遅延読み込み、MV画像の先行読み込み、JSの遅延読み込み・遅延実行などを実施し、最大でPSIスコアを28点向上させた
- PCSPで同じHTMLを使えるよう構造設計をし、実装をした
- 運用手順などをドキュメン化した
- その他にもレビュアーとしてコードレビューを行ったり、コードやGitまわりで困ったときに相談する人間としてメンバーのエラー解決を行った

#### 社内ハッカソン

ブログ記事投稿サイトの機能追加を対応した

- Laravel, Reactを用いてお気に入り機能、お気に入り記事一覧機能、記事評価機能の実装をした
