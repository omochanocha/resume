## 概要

フロントエンドエンジニアとして約4年の実務経験がある。UI/UXの実装だけでなく、Core Web Vitalsを指標としたパフォーマンス改善や運用保守性を考慮したコンポーネント設計に強みを持っている。また、チーム全体の生産性向上を意識し、ドキュメント整備や若手メンバーのサポート、技術記事の執筆など組織の技術力底上げにも貢献した。

## 自己PR

### 技術スキル

- **フロントエンド**： LPやアニメーションを伴うサイト、パフォーマンス要件の高いサイトなど、様々な要件のページをFigma等のデザインデータから設計・実装できる。またNext.js/React/TypeScriptを用いたコンポーネント設計・実装にも対応可能
- **バックエンド**： Laravel環境においてテーブル定義からコントローラー実装まで対応可能

### 開発における志向性・普段意識していること

- **保守・運用を見据えた設計**：将来の変更コストを抑えるため、共通化や抽象化の度合いを適切に判断しながらコードを書くことを意識している。あわせて、命名を丁寧に行うことや実装時の背景・他の選択肢を採用しなかった理由をコメントとして残すことで、後から読む人がコードの意図を追えるようにしている
- **型による安全性の担保**：TypeScriptの `satisfies` 演算子などを活用し、コンパイル時に網羅性チェックや実装漏れを検知できる型設計を行うようにしている
- **パフォーマンスへの意識**：Core Web Vitalsをはじめとした指標を起点に、ユーザー体験を損なわない実装を心がけている

### ソフトスキル

- **積極的な越境**：課題を解決するためには自分の領域外のことであっても周囲に助けを求めながら完遂できる。バックエンドの修正や画像の修正の経験があり、職種の垣根を超えてプロジェクトを前に進めることができる
- **チームメンバーへのサポート**： ドキュメント作成や新メンバーの環境構築、ツールの使い方、コーディングに不慣れなメンバーへのサポートを積極的に行い、チームへ貢献した

### 人材紹介サービスのデザイン部署におけるフロントエンド開発

期間：2022年7月~2025年3月

代表プロジェクト概要：複数領域でのデザインリニューアルとページスピードスコア改善

使用技術・ツール：HTML, CSS, JavaScript, Figma, Git/Github

#### 【主な取り組みと成果】

■ **Core Web Vitals改善によるUX向上**

○ 状況\
Page Speed Insightsのスコアが低下しており、改善依頼があった一方で、改善手順や具体的な施策が整理されていない状況だった。

■ **運用保守性を高める実装**\
要素が増減したときやテキストが複数行になったとき、アニメーションはどうするかなどのデザインデータに記載のない仕様の部分を事前にすり合わせ、手戻りのない実装になるよう主導した。

■ **積極的な越境**\
画像編集やバックエンド側の処理など専門外の領域でも、課題解決のために必要なことはデザイナーやバックエンドエンジニアの方にレビューをもらいながら対応した。

■ **チーム開発への貢献**\
